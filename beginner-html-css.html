Lesson 1: HTML Basics

HTML is the structure of a website. A simple HTML page looks like this:

<html>
<head><title>My First Page</title></head>
<body>
  <h1>Hello, World!</h1>
</body>
</html>

Lesson 2: CSS Basics

CSS is used to style HTML elements.

h1 {
  color: blue;
  text-align: center;
}
